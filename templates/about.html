About Hostel Visitor Log System

Hostel Visitor Log System is a web-based application developed using Python Flask. The main purpose of this project is to manage and monitor visitor records in a hostel efficiently. The system keeps track of visitor details such as Visitor ID, Name, Room Number, Entry Time, Exit Time, and Status. It helps hostel wardens maintain accurate records and improves hostel security. Features of the project:

  • Store visitor information
  • Track visitor entry and exit status
  • Display records in a tabular format
  • Easy navigation using Flask routes
  • Simple and user-friendly interface

This project is created for learning Python, Flask, HTML, Jinja2 templates, lists, dictionaries, loops, and web application development concepts.
